# GifsApp

En este proyecto se hace petición a un API externo con l fin de practicar conceptos como:
- Modularización de la aplicación
- Estructura de la aplicación en modulos y componentes
- Componentes
- ViewChild
- Servicios
- Historial de búsquedas
- Uso de Api Keys
- LocalStorage
- Peticiones HTTP
